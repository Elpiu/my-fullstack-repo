--- Set up appwrite self hosted

Go to the folder docker\1.8.0-appwrite
run -> docker compose up -d

---


--- Login and select user
appwrite client login --endpoint "http://localhost/v1"
appwrite login

--- Create project
appwrite init project


--- Generate types

appwrite types -l ts ./types


--- Other

appwrite client login --endpoint "http://localhost/v1"

appwrite client login --endpoint "https://cloud.appwrite.io/v1"

// Deplpoy cloud
Manually create on cloud appwrite project with same name and same id
Manually set auth settings (google oauth2)

appwrite push --all


--- Google oauth2
enable Google in appwrite console section auth

set app id and app secret from (https://console.cloud.google.com/)  Google Auth Platform / Client

In google Oauth 2 for the redirect Uri remuve 'fra'
https://fra.cloud.appwrite.io/v1/account/sessions/oauth2/callback/google/PROJECT_ID
https://cloud.appwrite.io/v1/account/sessions/oauth2/callback/google/PROJECT_ID

# aggiungere la tua app Angular come "Platform" nel progetto Appwrite con quel dominio per autorizzare il redirect
